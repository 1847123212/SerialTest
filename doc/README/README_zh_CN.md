# 串口助手
![downloads](https://img.shields.io/github/downloads/wh201906/SerialTest/total)  
跨平台串口测试工具  

[English](../../README.md)  

# 特点
+ 可在Windows, Ubuntu, Android, Raspbian上使用
+ 低内存占用（导出大量串口数据时很有用）
+ 实时绘图
+ 自定义控制面板
+ 支持Android蓝牙串口
